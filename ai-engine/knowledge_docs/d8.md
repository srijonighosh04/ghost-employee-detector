# Zero-Trust Rollout SOP

This Standard Operating Procedure details network security policies, bastion host requirements, and access controls.

## Authentication & Authorization
All API and console endpoints require Multi-Factor Authentication (MFA) via Okta. Session tokens are valid for a maximum of 12 hours before forcing re-authentication.

## Bastion Hosts
Bastion host configuration relies on Cloudflare Tunnel access lists:
- Direct SSH ports (22) on all workspace server instances are disabled.
- Developers must use `cloudflared access ssh` tunnels to establish secure connection terminals.

## Policy Exceptions & Gaps
We have a temporary exception list for legacy build servers (specifically the Jenkins builder on IP `10.104.22.45`) which does not support Okta MFA. It connects via pre-signed client certificates.
- **Critical Control**: If the client certificate is compromised, revoke the fingerprint in the bastion Access Control List (ACL) configuration immediately.
