# Growth Funnel Experiment Log

This log documents product experiments running on our signup and onboarding funnels.

## Experiment: 3-Step Sign-up Funnel (Exp-202)
- **Control Group (A)**: Original 5-step registration form (User, Profile, Company, Team, Integrations).
- **Treatment Group (B)**: Combined 3-step registration flow (User + Company, Team Invite, Integrations).
- **Target Metrics**: Increase sign-up completion rate by 15% without sacrificing company metadata completeness.

## Analytics Triggers
All onboarding events must dispatch structured JSON payloads to Segment:
- `onboarding_started`: Triggered when the user clicks 'Create Account'.
- `onboarding_step_completed`: Contains fields `step_name`, `step_index`, and `time_spent_seconds`.
- `onboarding_completed`: Dispatches when the company workspace database is successfully provisioned.
